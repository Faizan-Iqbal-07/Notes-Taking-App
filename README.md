# 📝 Notes Taking App

A simple Notes Taking App built using the MERN Stack.

## 🚀 Features

* Create Notes
* Read Notes
* Update Notes
* Delete Notes (CRUD)

## 🛠️ Tech Stack

### Frontend

* React.js
* Tailwind CSS
* Axios
* Context API

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose

## ⚙️ Installation

```bash
npm install
npm run dev
```

## 📌 Description

This application allows users to manage their notes easily. It uses Express.js for APIs, MongoDB with Mongoose for database operations, Axios for API handling, Context API for state management, and Tailwind CSS for responsive styling.
